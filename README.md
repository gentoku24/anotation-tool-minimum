# 3Dアノテーションツール

点群データに3Dバウンディングボックスを付けるためのアノテーションツールです。

## 機能

- 点群データの読み込みと表示
- 3Dバウンディングボックスの作成、編集、削除
- アノテーションの保存とロード
- クラスラベルの管理

## インストール方法

1. リポジトリをクローン
2. 依存パッケージのインストール
```
pip install -r requirements.txt
```

## 使用方法

```
python src/main.py
```

## データ構造

- `/data` - 点群データと保存されたアノテーションが格納されます
- `/class_labels` - クラスラベルの定義が格納されます
- `/src` - ソースコードが格納されます
